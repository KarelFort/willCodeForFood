1. Download eclipse IDE for java EE: https://eclipse.org/downloads/packages/eclipse-ide-java-ee-developers/lunasr2

2. Download Tomcat 8 (core binary): https://tomcat.apache.org/download-80.cgi
    2a. Unzip somewhere
    2b. Start eclipse
    2c. Go to window -> preferences -> server -> runtime env -> add -> choose path to tomcat

3. Download MySQL Community server: http://dev.mysql.com/downloads/mysql/ 
    -products to be installed: server, workbench, connector/ODBC, connector/J

4. Import sql dump file to MySQL: 
    Data Import/export -> Import from self-contained file - choose path to *.sql, create new schema -> Start import
